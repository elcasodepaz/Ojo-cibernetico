<h3 style="color: red;">Installation</h3>
<h4>Kali Linux / Parrot OS / Termux</h4>
<pre><code>
git clone https://github.com/elcasodepaz/Ojo-Cibernetico.git
cd Ojo-Cibernetico/
pip3 install -r requirements.txt
python3 Ojo-Cibernetico.py
</code></pre>

<h4>OSX</h4>
<pre><code>
git clone https://github.com/elcasodepaz/Ojo-Cibernetico.git
cd Ojo-cibernetico/
pip3 install -r requirements.txt
python3 Ojo-Cibernetico.py
</code></pre>
